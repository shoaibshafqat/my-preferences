== Description ==

My Preferences Enables a woocommerce customer to view a list of items and their category and allow him/her to create preferences for those categories and/or items in the list

= Main Features =

* Backend where from you can set custom image link to upload images.
* Import Categories and items from backend.
* After install auto page create which will show current logged in users preferences.

== Installation ==
1. Upload "woocommerce-my-preferences" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
